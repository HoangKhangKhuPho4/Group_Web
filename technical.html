Comprehensive Guide to Using GitHub

How to Create a Repository on GitHub

Follow these steps to create a new repository on GitHub:

  1. Go to the GitHub website and log in using your credentials.

  2. Click on the + (plus icon) in the top-right corner of the page and select New Repository from the dropdown menu.

  3. Fill out the repository details:

    • Repository Name: Enter a unique name that reflects the purpose of your project.
    • Description (optional): Provide a brief overview or purpose of the repository.
    • Visibility: Choose Public if you want everyone to access it or Private to restrict access.

  4. Optional: You can initialize the repository with:

    • A README file: Useful for providing an introduction or documentation.
    • A license template: To specify the terms under which your project can be used.
    • A .gitignore template: To exclude specific files from version control.

  5. Click the Create Repository button to finalize the process.

How to Create a GitHub Pages Site

Turn your repository into a static website using GitHub Pages by following these steps:

  1. Navigate to the repository you want to use for your GitHub Pages site.

  2. Click on the Settings tab at the top of your repository page.

  3. Scroll down to find the Pages section, typically located under the Code and automation category.

  4. Under Source, select the branch you want to use for GitHub Pages (e.g., main). You can also choose the folder to serve content from, such as root.

  5. Click Save. GitHub will generate a public URL for your site (e.g., https://your-username.github.io/your-repository-name).

  6. Once your site is published, add content (e.g., HTML, CSS, JavaScript) to your repository. These files will be displayed on your GitHub Pages site.

For more details, visit the official GitHub Pages documentation.
